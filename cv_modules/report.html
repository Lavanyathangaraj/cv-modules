Object Boundary Detection

This table reports the evaluation results for the Object Boundary Detection feature from Module 3, comparing the detected boundaries against across 10 distinct variations.

Metrics Used:

  • IoU (Jaccard Index): Measures the overlap accuracy between the detected boundary mask and the manually defined Ground Truth mask. (Goal: Closer to 1.0)
  • SSIM (Structural Similarity Index): Measures perceptual image quality between the original image and the final image with the drawn boundary. (Goal: Closer to 1.0)
  • Compute Time (ms): Measures the execution time for the script to generate the boundary.
